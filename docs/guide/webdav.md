# WebDav

### **WebDAV config**

| Name     | Value                       |
| -------- | --------------------------- |
| Url      | http[s]\://domain:port/dav/ |
| Host     | domain                      |
| Path     | dav                         |
| Scheme   | http/https                  |
| Port     | Same as web port            |
| Username | Same as web username        |
| Password | Same as web password        |

Can't fill in? [Click to view Fill in Example](#webdav-fill-in-example)
